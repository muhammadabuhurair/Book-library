# 📚 Book Library Website

A modern multi-page Book Library Website built using HTML and CSS. This project provides an organized platform where users can browse books, explore authors, learn about the library, and contact the library team.

---

## 🌐 Live Demo

🔗 Live Website: https://muhammadabuhurair.github.io/Book-library

---

## 📂 GitHub Repository

🔗 Repository: https://github.com/muhammadabuhurair/Book-library

## ✨ Features

### 🏠 Home Page

- Attractive hero section
- Featured books section
- Library statistics
- Categories section
- Responsive navigation menu

### 📖 Books Page

- Book collection display
- Search bar UI
- Book cover images
- Read More buttons

### 👨‍💼 Authors Page

- Author profile cards
- Author images
- Book information
- Professional card layout

### ℹ️ About Page

- Library introduction
- Mission section
- Library statistics
- Why choose our library section

### 📞 Contact Page

- Contact form
- Email information
- Phone number display
- Clean and simple layout
